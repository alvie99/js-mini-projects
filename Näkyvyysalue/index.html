Testaa näkyvyysaluetta

Tässä tehtävässä saat tarkasteltavaksi koodia, jossa on myös aivan uusia asioita. Eli näitä ei ole tarkoituskaan vielä tietää. Tutki kuitenkin koodia ja päättele mikä koodissa tekee mitäkin.
Jos koodi menee sotkuun, voit avata linkin uudelleen. Ohjelma päivittyy tasaisin väliajoin.

  1. Avaa konsoli eli console, se löytyy vasemmasta alareunasta.
  2. Ota JS-koodi näkyviin. HTML ja CSS ikkunat voit pienentää.
  3. Tarkastele JavaScript-koodia. Tutki mikä koodissa vastaa seuraavia kohtia:
    1. // Globaali muuttuja määritellään
    2. // Funktio aloitetaan
    3. // Funktiomuuttuja määritellään
    4. // Funktiomuuttuja tulostetaan
    5. // Funktiossa määritellään globaalin muuttujan arvon muutos, kun funktiota kutsutaan.
    6. // Funktion määrittely päättyy
    7. // Globaalin funktion arvo tulostetaan
    8. // Funktiota scopeTesti kutsutaan, joka tulostaa funktiomuuttujan ja muuttaa globaalin muuttujan arvoa
    9. // Globaalin muuttujan arvo tulostetaan uudelleen
  4. Kopio edellisestä listasta jokainen kohta (muista sisällyttää myös // merkit rivin alusta) ja lisää ne kommenteiksi JavaScriptiin sille kohdalle, johon ne kuuluvat. Eli copy-pasteta ne osaksi JS-ikkunan koodia.
  5. Muokkaa paikallisen funktiomuuttujan arvoa sanomaan "minut muokkasi _______" jossa viivat ovat sinun nimesi.
  6. Kopioi ja lisää koodin loppuun tämä pätkä koodia: console.log(funktioMuuttuja);
  7. Ota kuvakaappaus tästä sivusta ja vastaa Itsissä tehtävän kysymykseen: Miksi ohjelma kaatui?
